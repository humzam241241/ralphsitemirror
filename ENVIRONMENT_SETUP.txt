## 📋 Environment Variables You Need to Configure

### 1️⃣ **OpenAI API Key** (REQUIRED)
- **Where**: `backend/.env`
- **Variable**: `OPENAI_API_KEY=sk-proj-...`
- **Get it**: https://platform.openai.com/api-keys
- **Steps**: Sign in → Create new secret key → Copy key
- **Cost**: ~$5-20/month

### 2️⃣ **Supabase Database** (REQUIRED)
- **Where**: `backend/.env`
- **Variable**: `DATABASE_URL=postgresql://...`
- **Get it**: https://supabase.com
- **Steps**: 
  1. Create project
  2. Enable vector extension: `CREATE EXTENSION vector;`
  3. Run `backend/schema.sql`
  4. Run `backend/migrations/001_add_bookings_and_lead_enhancements.sql`
  5. Copy connection pooling URL
- **Cost**: Free (500MB)

### 3️⃣ **Gmail SMTP** (REQUIRED)
- **Where**: `backend/.env`
- **Variables**:
  ```bash
  SMTP_USER=your-email@gmail.com
  SMTP_APP_PASSWORD=16-char-password
  NOTIFICATION_EMAIL=info@ryansroofing.ca
  ```
- **Get it**: https://myaccount.google.com/apppasswords
- **Steps**: 
  1. Enable 2FA on Gmail
  2. Create app password (Mail → Other)
  3. Copy 16-character password (no spaces)
- **Cost**: Free

### 4️⃣ **Admin Secret** (REQUIRED)
- **Where**: `backend/.env` AND `admin/.env.local`
- **Variable**: `ADMIN_SECRET=your-32-char-secret`
- **Generate**: 
  - Mac/Linux: `openssl rand -base64 32`
  - Windows: `[Convert]::ToBase64String((1..32|%{Get-Random -Max 256}))`
- **⚠️ Must be the same in both files!**

### 5️⃣ **Cal.com API** (OPTIONAL - for bookings)
- **Where**: `backend/.env`
- **Variables**:
  ```bash
  CAL_COM_API_KEY=cal_live_...
  CAL_COM_EVENT_TYPE_ID=123456
  ```
- **Get it**: https://cal.com
- **Steps**: 
  1. Create event type
  2. Get API key from Settings → Developer
  3. Get event ID from URL
- **Cost**: Free
- **Skip if**: You don't need online booking yet

## 🚀 Quick Start Commands

After setting up environment variables:

```bash
# Start all services (Windows)
start.bat

# OR manually:
cd backend && npm install && npm run dev
cd frontend && npm install && npm run dev
cd widget && npm install && npm run dev
cd admin && npm install && npm run dev
```

**Services will run on**:
- Backend: http://localhost:8000
- Frontend: http://localhost:5173
- Widget: http://localhost:5174
- Admin: http://localhost:3001
